# Equipment List Tests

## Overview

Tests for the equipment list page (`/equipment`), including search, filters, pagination, and data display.

## Test Files

- `equipment-list.spec.ts` - Core list functionality
- `equipment-filters.spec.ts` - Filter controls and interactions
- Additional group tests in subdirectories

## Running

```bash
pnpm --filter frontend exec npx playwright test features/equipment/list
```
