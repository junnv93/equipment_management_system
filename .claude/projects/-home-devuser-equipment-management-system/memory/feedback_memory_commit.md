메모리 파일(.claude/projects/*/memory/)도 커밋 대상이다. 1인 프로젝트이므로 다른 PC에서도 같은 메모리를 봐야 한다.

**Why:** PC 이동 시 이전 세션의 피드백/프로젝트 메모리가 유실됨.

**How to apply:** 세션 마무리 시 메모리 파일 변경이 있으면 함께 커밋+푸시.
