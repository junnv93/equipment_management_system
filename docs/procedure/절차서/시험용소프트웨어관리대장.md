## 시스템 연동 참고사항

### 관리번호 체계

- 형식: `PNNNN` (P + 4자리 일련번호)
- 시험소 접두어: `SUW-` (수원), `UIW-` (의왕) 가능
- 현재 최대 번호: P0073

### 시험분야 분류

| 코드 | 설명                               |
| ---- | ---------------------------------- |
| RF   | Radio Frequency 시험               |
| SAR  | Specific Absorption Rate 시험      |
| EMC  | Electromagnetic Compatibility 시험 |
| RED  | Radio Equipment Directive 시험     |
| HAC  | Hearing Aid Compatibility 시험     |

### 가용 여부 상태

- **가용**: 현재 사용 가능한 유효한 소프트웨어
- **불가**: 버전 업그레이드 또는 교체로 인해 더 이상 사용하지 않는 소프트웨어

### 자체 유효성 검증

- **X**: UL-QP-18-09 양식에 따른 유효성 확인 대상
- **O**: 유효성 확인 미대상 (극소수, P0050만 해당)
