# 시험 소프트웨어의 유효성확인 (UL-QP-18-09)

> **양식번호**: UL-QP-18-09(02)  
> **관련 절차서**: UL-QP-18 장비 관리 절차서, 14. 소프트웨어의 유효성 확인  
> **원본 형식**: Word (.docx)  
> **용도**: 시험 소프트웨어가 정확한 결과를 산출하는지 검증하는 유효성 확인 기록

---

## 양식 구조

유효성 확인은 **두 가지 방법** 중 하나로 수행:

---

### 방법 1: 공급자 시연 (Demonstration by Vendor)

공급자가 소프트웨어의 유효성을 검증한 경우, 그 기록을 수령하여 대체.

| 항목                   | 영문                                                  | 설명                 |
| ---------------------- | ----------------------------------------------------- | -------------------- |
| 소프트웨어 업체명      | Software Vendor                                       | 소프트웨어 공급 업체 |
| 소프트웨어명 또는 설명 | Software Name or Description                          | 소프트웨어 식별 정보 |
| 소프트웨어 버전/개정일 | Software version and/or revision date                 | 버전 및 날짜         |
| 입수 일자              | Date of information                                   | 정보를 입수한 날짜   |
| 공급자 제공 정보 요약  | Summary of information provided by vendor             |                      |
| 수령한 UL 직원         | UL Staff Member Receiving Information                 |                      |
| 관련 정보 수령일       | Date on Which All Needed Information In Hand          |                      |
| 첨부 자료              | Scan and/or attach information provided by the vendor | 스캔본/첨부          |

---

### 방법 2: UL 자체 시험 (Validation tests performed by UL)

UL이 직접 유효성확인 시험을 수행하는 경우.

#### 기본 정보

| 항목                       | 영문                                  | 설명 |
| -------------------------- | ------------------------------------- | ---- |
| 소프트웨어명 또는 상세사항 | Software Name or Description          |      |
| 소프트웨어 제작자          | Software Author                       |      |
| 소프트웨어 버전/수정일     | Software Version and/or revision date |      |
| 참고 문서                  | Reference documents                   |      |
| 조작 단위 설명             | Operating unit description            |      |
| 소프트웨어 구성            | Software components                   |      |
| 하드웨어 구성              | Hardware components                   |      |

#### 획득 기능 검증 (Acquisition Functions)

| 항목                                          | 설명                                           |
| --------------------------------------------- | ---------------------------------------------- |
| 기능 명칭 또는 상세사항                       | 검증할 획득 기능                               |
| 소프트웨어와 별개로 시험 데이터를 획득한 방법 | 독립적 데이터 획득 방법 기술                   |
| 수락 기준                                     | 합격/불합격 판정 기준                          |
| 첨부                                          | 소프트웨어 획득 데이터 + 독립 획득 데이터 비교 |

> 획득 기능이 사용자 입력에 따라 달라지는 경우, 수차례 충분히 시험을 반복

#### 프로세싱 기능 검증 (Processing Functions)

| 항목                                            | 설명                                        |
| ----------------------------------------------- | ------------------------------------------- |
| 기능 명칭 또는 상세사항                         | 검증할 프로세싱 기능                        |
| 소프트웨어와 별개로 프로세스된 시험 데이터 방법 | 독립적 처리 방법 기술                       |
| 수락 기준                                       | 합격/불합격 판정 기준                       |
| 첨부                                            | 입력 데이터 + SW 출력 + 독립 처리 출력 비교 |

#### 제어 기능 검증 (Control Functions)

| 항목                                 | 설명                             |
| ------------------------------------ | -------------------------------- |
| 제어된 장비 기능                     | Equipment function controlled    |
| 예상되는 장비 기능                   | Expected equipment function      |
| 확인되는 장비 기능                   | Observed equipment function      |
| 장비 기능 결정 방법 (제어 SW와 무관) | 독립적 확인 방법                 |
| 수락 기준                            | 합격/불합격 판정 기준            |
| 첨부                                 | 로우 데이터, 확인사항, 관련 정보 |

#### 최종 확인

| 항목              | 영문                                                        | 설명 |
| ----------------- | ----------------------------------------------------------- | ---- |
| 시험 일자         | Date of Test                                                |      |
| 시험 수행자       | UL Staff Member Performing Test                             |      |
| 유효성확인 승인자 | UL Staff Member Authorizing Test as a Successful Validation |      |
