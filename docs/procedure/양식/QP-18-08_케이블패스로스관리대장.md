# Cable and Path Loss 관리대장 (UL-QP-18-08)

> **양식번호**: UL-QP-18-08(00)  
> **관련 절차서**: UL-QP-18 장비 관리 절차서  
> **원본 형식**: Excel (.xlsx)  
> **용도**: RF 시험용 케이블 및 경로 손실(Path Loss) 측정값 관리  
> **시트 구성**: RF Conducted (목록) + 개별 케이블 시트 (23개)

---

## 양식 구조

### 시트 1: RF Test Cable 목록

| #   | 컬럼명           | 설명                  | 예시                      |
| --- | ---------------- | --------------------- | ------------------------- |
| 1   | No               | 케이블 순번           | 1, 2, 3...                |
| 2   | 관리번호         | 케이블 고유 식별번호  | E020K-325, E100S-249      |
| 3   | Length (M)       | 케이블 길이 (미터)    | 0.2, 0.5, 1, 3            |
| 4   | TYPE             | 커넥터 타입           | K, SMA                    |
| 5   | 사용 주파수 범위 | 사용 가능 주파수 대역 | 30 MHz to 40 GHz          |
| 6   | S/N              | 시리얼 번호           | N/A                       |
| 7   | 위치             | 설치/보관 위치        | RF Shield room, Chamber 4 |

### 시트 2~N: 개별 케이블 Path Loss 데이터

각 케이블별 주파수-손실값 측정 데이터:

| 컬럼      | 설명                     | 단위      |
| --------- | ------------------------ | --------- |
| Freq(MHz) | 측정 주파수              | MHz       |
| Data(dB)  | 해당 주파수에서의 손실값 | dB (음수) |

---

## 관리번호 명명 규칙

```
ELLLX-NNN
│││  │
││└──┴── 커넥터 타입: K=K-type, S=SMA
│└────── 케이블 길이 (cm): 020=0.2m, 050=0.5m, 100=1m, 150=1.5m, 300=3m
└─────── E (Equipment 접두어)
```

## 데이터 예시

**케이블 목록 (상위 5건):**

| No  | 관리번호  | Length | TYPE | 주파수 범위      | 위치           |
| --- | --------- | ------ | ---- | ---------------- | -------------- |
| 1   | E020K-325 | 0.2    | K    | 30 MHz to 40 GHz | RF Shield room |
| 2   | E020S-322 | 0.2    | SMA  | 30 MHz to 27 GHz | RF Shield room |
| 3   | E020S-323 | 0.2    | SMA  | 30 MHz to 27 GHz | RF Shield room |
| 4   | E030K-327 | 0.3    | K    | 30 MHz to 40 GHz | RF Shield room |
| 5   | E030S-328 | 0.3    | SMA  | 30 MHz to 27 GHz | RF Shield room |

**Path Loss 측정값 예시 (E020K-325):**

| Freq(MHz) | Data(dB) |
| --------- | -------- |
| 30        | -0.041   |
| 100       | -0.074   |
| 500       | -0.153   |
| 1000      | -0.198   |
| 5000      | -0.342   |
| 10000     | -0.537   |
| 17000     | -0.709   |

> 주파수가 높을수록 손실이 커지는 것이 정상 패턴.
