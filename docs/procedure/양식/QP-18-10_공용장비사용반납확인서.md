# 공용장비 사용/반납 확인서 (UL-QP-18-10)

> **양식번호**: UL-QP-18-10(01)  
> **관련 절차서**: UL-QP-18 장비 관리 절차서  
> **원본 형식**: Word (.docx)  
> **용도**: 공용장비(여러 팀이 공유하는 장비)의 사용 대출 및 반납 기록

---

## 양식 구조

### Part 1: 공용 장비 사용 확인서

#### 사용 정보

| 항목        | 설명             |
| ----------- | ---------------- |
| 작성 (서명) | 확인서 작성자    |
| 승인 (서명) | 사용 승인자      |
| 사용부서    | 사용 요청 부서   |
| 사용자      | 실제 사용자 이름 |
| 사용장소    | 장비 사용 위치   |
| 사용기간    | 사용 예정 기간   |
| 사용목적    | 사용 목적/사유   |

#### 장비 목록

| 컬럼                | 설명                |
| ------------------- | ------------------- |
| 순번                | 장비 순번 (1~5)     |
| 장비명              | 장비 이름           |
| 모델명              | 장비 모델 번호      |
| 수량                | 사용 수량           |
| 관리번호            | 장비 관리번호       |
| 상태확인 - 외관     | 외관 상태 확인      |
| 상태확인 - 교정여부 | 교정 유효 상태 확인 |

#### 사용 확인

```
아래 목록과 같이 공용장비 사용(반출)을 확인합니다.
20   년    월   일
반출자 :                        (서명)
```


## 워크플로우

```
사용 요청 → 장비목록 작성 → 외관/교정 상태확인 → 승인 → 반출자 서명
→ (사용) →
반납 → 외관/이상여부 확인 → 특기사항 기록 → 반납자 서명 → 승인
```

## QP-18-06(장비반출입확인서)과의 차이

| 구분      | QP-18-06 반출입확인서 | QP-18-10 공용장비 확인서 |
| --------- | --------------------- | ------------------------ |
| 대상      | 외부 반출 장비        | 내부 공용 장비           |
| 이동 범위 | 시험소 외부           | 시험소 내부 (팀 간)      |
| 주요 확인 | 반출/반입 상태        | 외관/교정여부 + 이상여부 |
| 사유      | 교정, 수리, 렌탈      | 시험 목적 사용           |
