# 장비 반·출입 확인서 (UL-QP-18-06)

> **양식번호**: UL-QP-18-06(01)  
> **관련 절차서**: UL-QP-18 장비 관리 절차서  
> **원본 형식**: Word (.docx)  
> **용도**: 장비 외부 반출 및 반입 시 상태 확인 기록

---

## 양식 구조

### 1. 반출 정보

| 항목        | 설명                            |
| ----------- | ------------------------------- |
| 작성 (서명) | 반출 확인서 작성자              |
| 승인 (서명) | 반출 승인자                     |
| 반출지      | 반출 목적지                     |
| 전화번호    | 반출지 연락처                   |
| 반출주소    | 반출지 주소                     |
| 반출사유    | 반출 사유 (교정, 수리, 렌탈 등) |

### 2. 장비 목록 테이블

| 컬럼            | 설명                   |
| --------------- | ---------------------- |
| 순번            | 장비 순번 (1~14)       |
| 장비명          | 장비 이름              |
| 모델명          | 장비 모델 번호         |
| 수량            | 반출 수량              |
| 관리번호        | 장비 관리번호          |
| 상태확인 - 반출 | 반출 시 장비 상태 확인 |
| 상태확인 - 반입 | 반입 시 장비 상태 확인 |

### 3. 반출 확인

```
아래 목록과 같이 측정장비를 반출하였음을 확인합니다.
YYYY . MM . DD .
반출자 :                        (서명)
```

### 4. 특기사항

자유 기술란 — 특이사항, 주의사항 등 기록

### 5. 반입 확인

```
상기 목록과 같이 측정장비를 이상없이 반입하였음을 확인합니다.
YYYY . MM . DD .
반입자 :                        (서명)
```

### 6. 반입 결재

| 항목        | 설명               |
| ----------- | ------------------ |
| 작성 (서명) | 반입 확인서 작성자 |
| 승인 (서명) | 반입 승인자        |

---

## 워크플로우

```
반출 요청 → 장비목록 작성 → 반출 상태확인 → 승인 → 반출자 서명
→ (외부 이동) →
반입 → 반입 상태확인 → 특기사항 기록 → 반입자 서명 → 승인
```
