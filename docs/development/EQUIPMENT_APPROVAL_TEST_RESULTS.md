#### 통과한 테스트 (10개)

1. ✅ 시스템 관리자는 장비를 직접 승인할 수 있어야 합니다
2. ✅ 기술책임자가 요청을 승인할 수 있어야 합니다
3. ✅ 기술책임자가 요청을 반려할 수 있어야 합니다 (반려 사유 필수)
4. ✅ 파일을 업로드할 수 있어야 합니다
5. ✅ 파일 크기 제한을 초과하면 업로드가 실패해야 합니다
6. ✅ 허용되지 않은 파일 형식은 업로드가 실패해야 합니다
7. ✅ 시험실무자가 장비 수정 요청을 제출할 수 있어야 합니다
8. ✅ 시험실무자가 장비 삭제 요청을 제출할 수 있어야 합니다
9. ✅ 시스템 관리자는 장비를 직접 삭제할 수 있어야 합니다
10. ✅ 필수 필드가 누락되면 장비 등록이 실패해야 합니다

#### 개선이 필요한 테스트 (4개)

1. ⚠️ 시험실무자가 장비 등록 요청을 제출할 수 있어야 합니다

   - **원인**: 필수 필드 검증 실패 가능성
   - **상태**: 400 에러 발생 (검증 로직 확인 필요)

2. ⚠️ 기술책임자는 승인 대기 요청 목록을 조회할 수 있어야 합니다

   - **원인**: 권한 검증 또는 응답 형식 문제
   - **상태**: 400 에러 발생 (응답 형식 확인 필요)

3. ⚠️ 시험실무자는 승인 대기 목록을 조회할 수 없어야 합니다

   - **원인**: 권한 검증 로직 확인 필요
   - **상태**: 예상과 다른 상태 코드 반환

4. ⚠️ 추가 필수 필드가 포함된 장비 등록이 성공해야 합니다
   - **원인**: 필수 필드 검증 로직 확인 필요
   - **상태**: 400 에러 발생 (검증 로직 확인 필요)
