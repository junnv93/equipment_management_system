# 승인 워크플로우 (Approval Workflow)

UL-QP-18 절차서에 따른 다단계 승인 프로세스입니다. 기능별로 1-step, 2-step, 3-step 승인을 지원합니다.

## 역할 계층 (4단계)

| 역할                     | 한국어     | 레벨 | 승인 권한        |
| ------------------------ | ---------- | ---- | ---------------- |
| `test_engineer` (TE)     | 시험실무자 | 1    | 신청/등록만      |
| `technical_manager` (TM) | 기술책임자 | 2    | 1차 승인/반려    |
| `quality_manager` (QM)   | 품질책임자 | 3    | 검토 (교정 계획) |
| `lab_manager` (LM)       | 시험소장   | 4    | 최종 승인        |

## 승인 유형별 적용

### 1-Step 승인 (TM 승인)

```
신청 (TE) → TM 승인/반려 → 완료
```

| 대상 기능               | 설명                     |
| ----------------------- | ------------------------ |
| 반출 (Checkout)         | 교정/수리/대여 반출 승인 |
| 반출 반입 (Return)      | 반입 검사 결과 승인      |
| 교정 기록 (Calibration) | 교정 결과 승인           |
| 소프트웨어 (Software)   | 소프트웨어 변경 승인     |
| 반입 (Equipment Import) | 외부 장비 반입 승인      |

### 2-Step 승인 (TM 검토 → LM 승인)

```
신청 (TE) → TM 검토/반려 → LM 최종 승인/반려 → 완료
```

| 대상 기능            | 설명           |
| -------------------- | -------------- |
| 장비 폐기 (Disposal) | 장비 폐기 처리 |

### 3-Step 승인 (TM 제출 → QM 검토 → LM 승인)

```
작성 (TM) → QM 검토/반려 → LM 최종 승인/반려 → 완료
```

| 대상 기능                    | 설명             |
| ---------------------------- | ---------------- |
| 교정 계획 (Calibration Plan) | 연간 교정 계획서 |

## 통합 승인 관리

모든 승인 요청을 단일 화면에서 관리합니다.

```
승인 대기 목록:
  - 유형별 필터 (반출, 교정, 폐기, 교정 계획 등)
  - 역할별 자동 필터 (본인이 승인 가능한 건만 표시)
  - 승인/반려 처리
  - 승인 이력 조회
```

## 반려 처리

모든 승인 단계에서 반려 시:

- **반려 사유 필수** 입력
- 반려 단계 기록 (3-step의 경우 review/approval 구분)
- 신청자에게 알림 발송
- 재신청 가능 (수정 후 다시 제출)

## 직무 분리 (Segregation of Duties)

UL-QP-18에서 요구하는 직무 분리 원칙:

| 규칙                          | 설명                                |
| ----------------------------- | ----------------------------------- |
| 시험소장(LM)은 교정 등록 불가 | 승인 권한자가 직접 교정 수행 방지   |
| 신청자와 승인자 분리          | 본인이 신청한 건을 본인이 승인 불가 |
| 역할별 권한 제한              | RBAC + Permission Guard로 강제      |
